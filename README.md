TITLE
# AI Investor Agent – Multi-Agent Market Intelligence System


ABOUT
AI Investor Agent is a multi-agent system that transforms raw market data into actionable investment insights.
It detects market signals (bulk deals, breakouts), analyzes them using technical and contextual reasoning, and generates clear BUY/HOLD/WATCH recommendations.
The system also personalizes insights based on the user’s portfolio, prioritizing what matters most.


HOW IT WORKS
The system follows a multi-step agent pipeline:

1. Data Collection – fetches stock data, news, and market events  
2. Signal Detection – identifies patterns like breakouts or bulk deals  
3. Analysis – evaluates sentiment and technical indicators  
4. Decision – generates actionable recommendations  
5. Personalization – prioritizes results based on user portfolio  

All steps run autonomously without human intervention.


TECH STACK
- Python
- Streamlit
- yfinance
- pandas, numpy
- ta (technical indicators)
